## 🚨 CONSTRAINTS & SAFETY

✅ **Must have:**
- Explicit environment selection per run
- Production execution disabled by default
- All secrets masked in logs
- Unique test data prefixes
- Controlled cleanup (teardown guaranteed)
- Timeout per test (30s default)
- Retry only for idempotent calls (GET only)

❌ **Never:**
- AI hallucinating endpoints
- AI replacing deterministic tests
- Auto-blocking CI without review
- Fuzzing or mutation testing
- Uncontrolled parallel destructive tests
