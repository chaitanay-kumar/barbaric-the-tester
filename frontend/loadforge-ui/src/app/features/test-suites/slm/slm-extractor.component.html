🧠 AI Rule Extraction

Extract structured business rules from PR descriptions, task text, or documentation. The SLM identifies validation rules, constraints, and access patterns — then generates test cases flagged for human review.

ℹ️
All SLM-generated tests require review. They are flagged with needs_review: true and cannot gate CI by default. AI is an enhancer — not the foundation. 85-90% of coverage comes from deterministic generation.

Input Text

Paste a PR description, task requirements, API documentation, or any text describing business rules.
